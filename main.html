ABOUT OUR PROJECT
Hey Guys, in this project we are implementing what we know as zipper. So basically we are compressing our files, that is LOSS LESS compression. And Finally we will get a compressed or encoded file for our original text.
NOTE - Since our project is at a basic stage only, so it works well for only .txt file. For PDF and DOCX file it may not work properly.
FOR ENCODING :
Step 1 - Click on "CHOOSE FILE" button :- To choose the original text file which has to be compressed.



Step 2 - CLick on "Encode" button :- After choosing the file a huffman binary tree can be seen on the left pane of the window. This button COMPRESSES the original file and downloads it simultaneously.
Your file is Successfully compressed now.

FOR DECODING :
Step 1 - Click on "CHOOSE FILE" button :- To choose the original text file which has to be compressed.



Step 2 - CLick on "Decode" button :- After choosing the file a huffman binary tree can be seen on the left pane of the window. This button DECOMPRESSES the original file and downloads it simultaneously.
Your file is Successfully decompressed now.


The Compression Algorithm used in our project is HUFFMAN ENCODING.
We have used this algorithm to implement our Encoding and Decoding of the file.
